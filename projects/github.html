Evaluatie: Git & Github.

Beschrijving van de opdracht

Hier moesten we onze kennis van Github tonen aan de hand van een simpele html opdracht.

Wat is Github

Git is een versiebeheersysteem dat wordt gebruikt om wijzigingen in bestanden bij te houden en samen te werken aan projecten. GitHub is een online platform dat Git gebruikt om projecten te hosten en samen te werken aan code.

Doel van de Evaluatie:

Het doel van deze evaluatie was om zeker te zijn dat we de basisprincipes van Git en GitHub begrijpen. Dit was zeer belangrijk omdat elke PE in de toekomst via GitHub zal verlopen.

Wat ik heb geleerd:

  • Begrijpen van de basisprincipes van versiebeheer met Git.
  • Leren hoe je een repository maakt en beheert op GitHub.
  • Uitvoeren van commits, pushen en pullen van wijzigingen.
