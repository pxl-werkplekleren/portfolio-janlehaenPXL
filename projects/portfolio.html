Opdracht: Portfolio

Beschrijving van de opdracht

Een portfolio maken in HTML en CSS waar we een overzicht geven van alle projecten die we hebben gemaakt tijdens werkplekleren.

Doel van de opdracht:

Onze projecten en competenties presenteren in een professioneel en overzichtelijk portfolio die we kunnen gebruiken om onze vooruitgang te laten zien.

Wat ik heb geleerd:

  • Het programma Visual Code. We hebben webstorm gekregen voor tijdens onze studies, maar ik heb besloten om mijzelf vertrouwd te maken met Visual Code omdat dit waarschijnlijk de software is waar ik later ook mee ga werken.
  • Mijn kennis van HTML en CSS is aanzienlijk verbeterd.

Reflectie:

Dit is tot nu toe het leukste project dat ik heb gemaakt. Ik heb er veel tijd en aandacht in gestoken, en ik ben er oprecht trots op. Het proces was intensief, maar het resultaat voelt als een echte stap vooruit in mijn ontwikkeling.
